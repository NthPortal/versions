## Add as a Dependency

### SBT (Scala 2.11 and 2.12)
```
"com.nthportal" %% "versions" % "1.0.1"
```

### Maven

**Scala 2.12**

```
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>versions_2.12</artifactId>
  <version>1.0.1</version>
</dependency>
```

**Scala 2.11**

```
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>versions_2.11</artifactId>
  <version>1.0.1</version>
</dependency>
```

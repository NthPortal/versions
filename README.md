## Add as a Dependency

### SBT

**Scala 2.12**

```sbtshell
"com.nthportal" %% "versions" % "2.0.0"
```

**Scala 2.11**

```sbtshell
"com.nthportal" %% "versions" % "1.2.1"
```

### Maven

**Scala 2.12**

```xml
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>versions_2.12</artifactId>
  <version>2.0.0</version>
</dependency>
```

**Scala 2.11**

```xml
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>versions_2.11</artifactId>
  <version>1.2.1</version>
</dependency>
```
